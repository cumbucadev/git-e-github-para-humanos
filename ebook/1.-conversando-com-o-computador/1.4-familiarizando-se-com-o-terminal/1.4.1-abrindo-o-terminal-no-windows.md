# 1.3.1 Abrindo o terminal no Windows

No Windows, existem diferentes formas de acessar o terminal. Para quem está começando, o mais importante não é entender todas as diferenças técnicas, mas saber qual opção usar e como acompanhar os comandos apresentados ao longo do livro.

## Qual terminal usar no Windows

Sempre que possível, utilize o **Windows Terminal**. Essa é a opção mais moderna e recomendada atualmente.

Ao abrir o Windows Terminal, o ambiente que normalmente é iniciado é o **PowerShell**. Na prática, isso significa que você já estará pronta para executar os comandos usados neste livro, sem precisar fazer nenhuma configuração adicional.

Se o Windows Terminal não estiver disponível no seu computador, utilize o **PowerShell** diretamente.

O **Prompt de Comando (CMD)** é uma alternativa mais antiga e deve ser usado apenas se você não tiver acesso ao Windows Terminal nem ao PowerShell.

## Como abrir o terminal

A forma mais simples de abrir o terminal é usando a busca do próprio Windows.

Vá até a barra de tarefas, clique no campo de busca ou no ícone de lupa e comece a digitar **terminal**.

Podem aparecer opções como:

* Windows Terminal
* PowerShell
* Prompt de Comando

Se aparecer **Windows Terminal**, essa é a opção recomendada. Caso contrário, escolha **PowerShell**. Use **Prompt de Comando** apenas se nenhuma das opções anteriores estiver disponível.

Ao abrir o terminal, uma janela será exibida. Ela pode ter fundo claro ou escuro, dependendo das configurações do sistema. A aparência não é importante neste momento.

## Como ler os comandos deste livro

A partir daqui, os comandos apresentados indicarão para qual sistema eles devem ser usados.

Quando um comando funcionar da mesma forma no Prompt de Comando e no PowerShell, ele será indicado como:

**Windows (CMD/PowerShell)**

Isso significa que o comando pode ser executado normalmente em qualquer um dos dois.

Somente quando houver diferença entre os ambientes, o comando será indicado separadamente como:

* **Windows (PowerShell)**
* **Windows (CMD)**

Você não precisa se preocupar em identificar qual terminal está usando em detalhes. Basta observar a indicação que acompanha cada comando e seguir a instrução correspondente.

***

Se esta for a sua primeira vez utilizando um terminal, é normal que tudo pareça estranho no início. Por enquanto, o objetivo é apenas ter o terminal aberto.\
A partir daqui, você já está pronta para começar a executar seus primeiros comandos, com tranquilidade e passo a passo.
