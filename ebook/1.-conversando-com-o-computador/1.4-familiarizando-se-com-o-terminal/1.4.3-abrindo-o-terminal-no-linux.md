# 1.3.3 Abrindo o terminal no Linux

No Linux, a forma de abrir o terminal pode variar um pouco dependendo da distribuição e do ambiente gráfico que você utiliza. Ainda assim, existem caminhos bastante comuns.

Na maioria das distribuições, você pode usar a busca do sistema. Procure por um campo de busca ou menu de aplicativos e digite **Terminal**. Normalmente, o aplicativo aparece com esse nome.

Outra forma bastante comum é usar um atalho de teclado. Em muitas distribuições Linux, pressionar **Ctrl + Alt + T** abre o terminal diretamente. Se funcionar no seu sistema, uma janela de terminal aparecerá na tela.

Assim como nos outros sistemas, ao abrir o terminal no Linux, você verá uma janela com texto. Não é necessário entender tudo de imediato. O objetivo deste momento é apenas se familiarizar com a ferramenta.

Se o nome do aplicativo ou o caminho até ele for um pouco diferente no seu Linux, não se preocupe. O importante é saber que existe uma ferramenta chamada terminal e que ela pode ser aberta para receber comandos.

A partir de agora, com o terminal aberto, está tudo pronto para você começar a dar os seus primeiros comandos.
