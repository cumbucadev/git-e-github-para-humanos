# 1.4 Principais comandos

Agora que você já abriu o terminal, é hora de começar a enviar suas primeiras ordens ao computador.

Nesta seção, vamos conhecer os **principais comandos** que você vai usar ao longo deste livro. Eles cobrem **tudo o que será necessário** para acompanhar os exemplos, exercícios e explicações dos próximos capítulos. Nada aqui é extra ou avançado demais: cada comando tem um propósito claro e vai aparecer várias vezes mais adiante.

Não se preocupe em **decorar tudo agora**. Esse não é o objetivo. A ideia é criar **familiaridade**: reconhecer os nomes, entender para que cada comando serve e começar a se sentir confortável ao vê-los na tela. Com o tempo e, principalmente, com a prática, esses comandos deixam de parecer estranhos e passam a fazer parte do seu vocabulário natural.

Vale reforçar: você **sempre pode voltar a esta seção** para consultar os comandos novamente. Isso faz parte do processo. Mesmo profissionais experientes usam o terminal diariamente e, ainda assim, volta e meia precisam conferir um comando, uma opção ou um detalhe específico.

Pense nesta seção como um primeiro contato e também como um material de referência. Você não precisa dominar nada ainda — apenas entender o suficiente para saber _o que existe_ e _para que serve_. O domínio vem depois, passo a passo, conforme você usa o terminal de verdade.
