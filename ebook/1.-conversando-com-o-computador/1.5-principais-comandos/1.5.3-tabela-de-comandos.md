# 1.4.3 Tabela de comandos

Para facilitar a consulta e servir como material de referência rápida, a tabela a seguir reúne os principais comandos do terminal apresentados nesta seção. Ela mostra, lado a lado, os comandos equivalentes em Linux, macOS e Windows, além de indicar claramente o que cada comando faz e se há ou não saída após a execução.

| Ação                      | Linux / macOS          | Windows (CMD)             | Windows (PowerShell)      | O que faz                                       | Saída                      |
| ------------------------- | ---------------------- | ------------------------- | ------------------------- | ----------------------------------------------- | -------------------------- |
| Ver diretório atual       | `pwd`                  | `pwd`                     | `pwd`                     | Mostra o caminho completo da pasta atual        | Caminho do diretório atual |
| Listar arquivos e pastas  | `ls`                   | `dir`                     | `dir`                     | Lista arquivos e pastas do diretório atual      | Lista de arquivos e pastas |
| Entrar em uma pasta       | `cd nome-da-pasta`     | `cd nome-da-pasta`        | `cd nome-da-pasta`        | Altera o diretório atual para a pasta indicada  | Nenhuma                    |
| Voltar para a pasta acima | `cd ..`                | `cd ..`                   | `cd ..`                   | Sobe um nível na hierarquia de pastas           | Nenhuma                    |
| Ir para a pasta pessoal   | `cd ~`                 | `cd ~`                    | `cd ~`                    | Vai diretamente para a pasta pessoal do usuário | Nenhuma                    |
| Criar arquivo vazio       | `touch arquivo.txt`    | `type nul > arquivo.txt`  | `New-Item arquivo.txt`    | Cria um arquivo vazio no diretório atual        | Nenhuma                    |
| Criar pasta               | `mkdir pasta`          | `mkdir pasta`             | `mkdir pasta`             | Cria uma nova pasta                             | Nenhuma                    |
| Copiar arquivo            | `cp origem destino`    | `copy origem destino`     | `copy origem destino`     | Cria uma cópia de um arquivo                    | Nenhuma                    |
| Copiar pasta              | `cp -r origem destino` | `xcopy origem destino /E` | `xcopy origem destino /E` | Copia uma pasta e todo o seu conteúdo           | Nenhuma                    |
| Mover / renomear          | `mv origem destino`    | `move origem destino`     | `move origem destino`     | Move ou renomeia arquivos e pastas              | Nenhuma                    |
| Excluir arquivo           | `rm arquivo`           | `del arquivo`             | `del arquivo`             | Remove definitivamente um arquivo               | Nenhuma                    |
| Excluir pasta             | `rm -r pasta`          | `rmdir /S pasta`          | `rmdir /S pasta`          | Remove uma pasta e todo o seu conteúdo          | Nenhuma                    |
