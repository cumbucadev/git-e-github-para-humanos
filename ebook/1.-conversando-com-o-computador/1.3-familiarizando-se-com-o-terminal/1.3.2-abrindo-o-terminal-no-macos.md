# 1.3.2 Abrindo o terminal no MacOS

No macOS, o terminal já vem instalado por padrão e é fácil de encontrar.

Uma das formas mais simples de abrir é usando o **Spotlight**, a busca do macOS.

Pressione as teclas **Command (⌘) + Espaço** ao mesmo tempo. Um campo de busca vai aparecer no centro da tela. Digite a palavra **Terminal**.

Quando o aplicativo **Terminal** aparecer nos resultados, pressione **Enter** ou clique nele.

Outra forma de abrir o terminal é pelo Finder. Abra o Finder, vá até a pasta **Aplicativos**, depois entre na pasta **Utilitários**. Dentro dela, você encontrará o aplicativo **Terminal**. Clique duas vezes para abrir.

Assim que o terminal abrir, você verá uma janela com texto. Mesmo que não entenda tudo o que aparece ali, isso é normal. O importante é que o terminal está aberto e pronto para uso.
