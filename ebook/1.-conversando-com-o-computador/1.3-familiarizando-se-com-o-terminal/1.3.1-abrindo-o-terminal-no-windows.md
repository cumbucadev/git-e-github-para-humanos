# 1.3.1 Abrindo o terminal no Windows

No Windows, existem algumas formas diferentes de acessar o terminal. Aqui, vamos focar na maneira mais simples e comum para quem está começando.

A forma mais direta é usando a busca do próprio sistema.

Primeiro, vá até a barra de tarefas, normalmente localizada na parte inferior da tela. Clique no campo de busca ou no ícone de lupa. Em seguida, comece a digitar a palavra **terminal**.

Dependendo da versão do Windows, algumas opções podem aparecer:

* **Windows Terminal**
* **Prompt de Comando**

Se aparecer **Windows Terminal**, essa é a opção recomendada. Clique nela para abrir. Se não aparecer, ou se seu Windows for uma versão mais antiga, você pode clicar em **Prompt de Comando**.

Ao abrir, uma janela será exibida. Ela pode ser clara ou escura, dependendo das configurações do sistema. Independentemente da aparência, você estará no lugar certo.

Se for a primeira vez que você abre o terminal no Windows, não se preocupe se muitas coisas parecerem estranhas. Por enquanto, o objetivo é apenas conseguir abrir a ferramenta.
