# Solução Exercício  1

Aqui estão os comandos que você deve usar (lembre-se de substituir com seu nome e email reais):

```bash
git config --global user.name "Seu Nome Aqui"
git config --global user.email "seu@email.com"
```

Para verificar se a configuração foi aplicada corretamente, use:

```bash
git config --global --list
```

Isso deve mostrar algo como:

```bash
user.name=Seu Nome Aqui
user.email=seu@email.com
```

***

:books: Pronto! Agora o Git já sabe quem é você. Essas configurações são importantes para que seu nome e email apareçam corretamente nos commits, ajudando a identificar quem fez o quê em um projeto.
