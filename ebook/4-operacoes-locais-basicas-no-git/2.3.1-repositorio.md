# 4.1 O quê é um Repositório Git?

Um repositório (ou _repo_) é a maior unidade de armazenamento do Git. É o local onde ficam todos os seus códigos, arquivos e o histórico completo de alterações — como se fosse uma grande pasta que guarda não apenas os arquivos, mas também todos os registros de modificações e metadados.

O repositório pode existir em dois lugares: localmente, no seu computador, ou remotamente, em um servidor.

## Local

Um repositório **local** é aquele que está no seu computador. Ele permite que você:

* Trabalhe em seus arquivos mesmo sem conexão com a internet;
* Salve alterações, criar histórico de mudanças e organizar o desenvolvimento;
* Testar e experimentar sem afetar outros colaboradores.

Pense nele como uma **pasta privada**, mas com superpoderes: ela sabe **tudo o que já aconteceu em cada arquivo**.

## Remoto

Um repositório **remoto** é aquele hospedado em um servidor, Ele permite que:

* Você compartilhe seu trabalho com outras pessoas;
* Múltiplos colaboradores trabalhem no mesmo projeto;
* O repositório seja **público** (qualquer pessoa pode visualizar) ou **privado** (apenas pessoas autorizadas podem acessar).

Mesmo quando o código é público, para **alterar o repositório principal** você precisa de permissão. Isso garante que mudanças só sejam feitas por quem realmente tem autorização, conceito que veremos mais adiante.

<figure><img src="../.gitbook/assets/0432.png" alt=""><figcaption></figcaption></figure>



## Local e Remoto

O repositório local e o remoto estão conectados. Na prática:

* Você faz alterações no **local**;
* Depois, envia essas alterações para o **remoto;**
* Outros pessoas colaboradoras podem enviar alterações para o remoto, que você trará para o seu local.

> Pense no local como **sua área de trabalho pessoal**, e no remoto como **a versão central compartilhada do projeto**. Eles se sincronizam constantemente para manter o histórico completo atualizado.

Essa relação permite que você trabalhe **de forma independente**, mas ainda colabore com segurança com outras pessoas.
