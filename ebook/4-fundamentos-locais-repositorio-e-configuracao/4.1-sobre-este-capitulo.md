# 4.1 Sobre este capítulo

Este capítulo é o seu primeiro mergulho em **operações locais** no Git. Aqui você entende o que é um repositório de verdade. Você também prepara o seu ambiente para começar a versionar.

O foco é **100% no seu computador**. Sem GitHub. Sem repositório remoto. Essa base deixa os próximos capítulos mais simples.

Ao longo do capítulo, você vai:

* entender o que é um **repositório Git** e onde ele pode existir (local vs remoto);
* criar o seu primeiro repositório local com `git init`;
* entender por que a pasta `.git` é essencial (e por que não mexer nela);
* aprender como funcionam as **configurações locais** do Git;
* configurar sua identidade no Git (ex.: `user.name` e `user.email`).

Ao terminar este capítulo, você deve conseguir iniciar um projeto com Git do zero.
