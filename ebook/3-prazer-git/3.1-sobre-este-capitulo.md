# 3.1 Sobre este capítulo

Este capítulo é o seu primeiro contato prático com o **Git**. Aqui você sai da ideia de “controle de versão” e conhece a ferramenta em si.

Vamos começar entendendo o que é o Git, por que ele existe e por que virou padrão na indústria. Em seguida, você vai instalar o Git no seu computador, de acordo com o seu sistema operacional.

Depois da instalação, o foco muda para o uso no dia a dia. Você vai entender como interagir com o Git (GUI vs CLI) e por que, neste curso, vamos priorizar a **linha de comando**.

Por fim, você vai aprender duas coisas que te dão autonomia:

* como ler a estrutura de um comando `git` (comando, opções e argumentos);
* como pedir ajuda ao próprio Git usando `git help`, `--help` e `-h`.

Ao terminar este capítulo, você deve conseguir abrir o terminal, confirmar que o Git está instalado e navegar pela ajuda oficial sem travar.
